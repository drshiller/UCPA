To save this setup to disk for installation at another time:

  1. In the File Download box, select Save this program to disk. 

  2. Save the file named "setup.exe" to your hard drive. 

  3. When you're ready to install, locate setup.exe on your hard drive and double-click the file to begin installation. 

  4. The InstallShield Wizard will then guide you through the setup.
