Document Viewer Users Guide

You can view many of our records online using our powerful document viewer. Just look for the document icons next to a search result, and click on it to view the document.
 
The Document Viewer Button Bar
Settings: When selected, the Display Properties window will appear.
Fit In Window: Places the entire page, top to bottom, in the window for viewing.
Fill Window: Places the page in the window, left to right, for viewing.
Rotate Left: Rotates the image in a counter clockwise direction, 90 degrees at a time.
Rotate Right: Rotates the image in a clockwise direction, 90 degrees at a time.
Pan: With the left mouse button depressed, moving the mouse permits the user to move the page around in the window. (Panning and Zooming can't be used at the same time.)
Zoom: Expand a defined area of an image to fill the entire window.
Quick View: Quick access to the Multiple Image Display selection screen. Click here for more details..
First Page: In a multiple page document, the first page will be viewed.
Previous Page: In a multiple page document, permits moving back one page at a time.
Next Page: In a multiple page document, permits moving forward one page at a time.
Last Page: In a multiple page document, the last page will be viewed.
Go To Page: Permits the user to select a specific page to be viewed.
Click here for more details..
Print Document: Permits the user to print an image of a document.
Click here for more details..
About RVI: Provides information about the RVI document viewer.
 
The Document Viewer Status Bar
Ready Box: Indicates when the imaging system is connected and active.
Document Box: Reflects the number of pages in the document being viewed.
Image File Name: The file name of the current image being viewed.
